# Weg UI Component Showcase

A component showcase and development environment for Weg UI components built with React, Radix UI, and Tailwind CSS.

## Getting Started

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Start Storybook
pnpm storybook
```

## Tech Stack

- **React 19** - UI framework
- **TanStack Router** - Routing
- **Radix UI** - Accessible component primitives
- **Tailwind CSS** - Styling
- **Storybook** - Component documentation
- **Vite** - Build tool

## Components

Available components include Button, Alert, Accordion, Card, Checkbox, Dialog, Radio, Progress, Spinner, Tooltip, Tabs, Pagination, Carousel, and Badge.

View components in the app or explore them in Storybook for interactive documentation and examples.
